# Smart Room - Platform Pembelajaran Interaktif

Smart Room adalah platform pembelajaran interaktif yang dirancang untuk membantu pengguna memahami konsep Smart Home melalui simulasi visual yang mudah dipahami.

## Fitur Utama

- 🎮 **Simulasi Interaktif**: Belajar melalui permainan yang menyenangkan
- 🏠 **Ruangan Pembelajaran**: Berbagai skenario yang tersedia untuk dipelajari
- 📊 **Pelacakan Kemajuan**: Pantau perkembangan pembelajaran Anda
- 👥 **Sistem Login**: Akun pribadi untuk menyimpan kemajuan

## Cara Memulai

Untuk menjalankan aplikasi:

```bash
npm run dev
```

Buka browser dan akses [http://localhost:3000](http://localhost:3000)

## Struktur Aplikasi

- `app/` - Halaman utama aplikasi
- `components/` - Komponen yang dapat digunakan kembali
- `public/` - Aset dan gambar
- `styles/` - Gaya visual aplikasi

## Persyaratan Sistem

- Node.js versi 14 atau lebih tinggi
- Koneksi internet untuk mengakses layanan

Untuk informasi lebih lanjut, silakan hubungi tim pengembang.
